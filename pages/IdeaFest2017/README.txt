Souls - Submission for OSU 2017 Ideas Fest

About:
Souls is a smart, insertable sole which fits into any shoe to collect data on how you run, walk, and move to help you live healthier
 
Visit Live webpage: https://sweetmantech.github.io/pages/IdeaFest2017/app/index.html

Our Process: 
1) Conduct research to start creating the first pedobarographic data set
2) Create user interface in IOS and Android applications
3) Integrate the deep-learning model
